Odin Recipes

This project was created for the Odin Project as part of the Foundations curriculum.

The goal of this project was to build a simple, multi-page site to demonstrate basic adequacy in HTML structure.

I decided to thematically base my recipes in the fictional universe of Project Zomboid by The Indie Stone because
it is my favorite video game. I had a ton of fun making this project, I hope it makes you smile.
